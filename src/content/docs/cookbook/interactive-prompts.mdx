Some commands need to ask the user questions, show progress, or handle long-running work gracefully. Repl provides a typed interaction channel for all of these.

## Prompts

Inject `IInteraction` to ask the user questions mid-command:

```csharp
app.Map("add", async (IInteraction ui, IContactStore store) =>
{
    var name = await ui.Ask("Full name:");
    var email = await ui.Ask("Email:");

    var contact = store.Add(name, email);
    return contact;
});
```

In REPL mode this presents an interactive prompt. In CLI mode with `--answer:*` flags, answers are pre-filled deterministically.

## Confirmation

```csharp
app.Map("delete {id:int}", async (int id, IInteraction ui, IContactStore store) =>
{
    var contact = store.Get(id);
    var confirmed = await ui.Confirm($"Delete '{contact.Name}'? This cannot be undone.");

    if (!confirmed)
        return Results.Cancelled("Delete cancelled.");

    store.Delete(id);
    return Results.Success($"Deleted '{contact.Name}'.");
});
```

## Progress

```csharp
app.Map("import {file}", async (string file, IInteraction ui, CancellationToken ct) =>
{
    var lines = await File.ReadAllLinesAsync(file, ct);

    await using var progress = await ui.StartProgress($"Importing {lines.Length} records...");

    for (int i = 0; i < lines.Length; i++)
    {
        ct.ThrowIfCancellationRequested();
        await ImportLine(lines[i]);
        progress.Report((double)i / lines.Length);
    }

    return Results.Success($"Imported {lines.Length} records.");
});
```

## Timeouts

```csharp
app.Map("ping {host}", async (string host, IInteraction ui) =>
{
    using var cts = new CancellationTokenSource(TimeSpan.FromSeconds(5));
    try
    {
        var reply = await Ping(host, cts.Token);
        return $"{host}: {reply.RoundtripTime}ms";
    }
    catch (OperationCanceledException)
    {
        return Results.Error($"Ping to {host} timed out after 5s.");
    }
});
```

## Deterministic automation with `--answer:*`

Pre-fill prompts from the command line — useful for scripts and tests without `Repl.Testing`:

```bash
# Answers prompt 1 with "Alice", prompt 2 with "alice@example.com"
myapp add --answer:1=Alice --answer:2=alice@example.com
```

This makes interactive commands fully scriptable without changing handler code.

## Spectre.Console prompts

With `Repl.Spectre`, upgrade all prompts to rich Spectre.Console widgets automatically — selection lists, multi-select, styled confirmations:

```csharp
builder.Services.AddSpectreInteraction();
```

No handler changes required. See the [Spectre cookbook](/cookbook/spectre/) for details.
